To specify any of rules above use node types, described in Esprima's source files (e.g. IfStatement, ForStatement or CallExpression) divided by space.

Structure rule gives you ability to check roughly structure of the program in the following way: to specify structure, type appropriate node types divided by space in the order they are nested into each other.
For example, ForStatement IfStatement stands for «if statement inside the for loop». Example #4 shows visually how useful this could be.
